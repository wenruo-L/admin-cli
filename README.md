# patpet-admin-web

> Patpet App 自动抓拍图片清晰度管理页面与关键词测试页面 -暂时

## 项目开发

```javascript
// 安装依赖
yarn

// 开发环境
yarn build:dev

// 测试环境
yarn build:sit

// 预上线环境
yarn build:stage

// 生产环境
yarn build:prod

```

## 页面路径

生产环境域名：https://admin.patpet.com.cn/#/  
测试环境域名：https://test.admin.patpet.com.cn/#/

清晰度管理页面：/ (默认页面)  
关键词测试页面：/stickers/stickers-keyword-test

## 打包后路径

/dist
